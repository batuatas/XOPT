# Scenario Flagship Redesign Brief — v4

## What Failed In The Existing Scenario Graphics

The current scenario explainer package is directionally useful, but the visual system is not strong enough for main-stage conference use.

The main failures are structural:

- too many figures are built from repeated cards, boxes, or column panels rather than one dominant visual argument
- the search-method visual reads like a cartoon process chart instead of a selective research funnel
- the question/objective figure is a row of boxed templates rather than an analytic comparison
- the plausibility figure is too sparse: pale intervals plus dots, with weak density and weak hierarchy
- the regime-to-portfolio figure is fragmented into three separate mini-arguments instead of one transition story
- several scenario visuals look assembled from components rather than editorially composed

## What To Remove

- box-heavy flowchart layouts
- card-grid question summaries
- thin interval plots that leave most of the canvas empty
- multi-panel collages where no single structure clearly dominates
- any main-stage dependence on legacy MALA framing

## What To Keep Only As Content

- the reset pipeline stages
- the surviving question set
- the actual selected scenario outputs
- the macro plausibility logic
- the regime-transition interpretation
- the portfolio consequences

These survive as content, not as layout templates.

## New Visual Logic

The replacement package will use one editorial structure per figure:

1. a true narrowing funnel for the search process
2. a tight analytic row structure for question and objective design
3. a dense plausibility graphic built from historical distributions, anchor markers, and selected scenario clusters
4. a directional regime-transition visual built around before/after dimension scores plus the main macro drivers
5. a single allocation-consequence figure built around baseline versus scenario portfolio shifts

## Content Selection Decision

Only three scenario stories are strong enough for the flagship package:

- `Q1_gold_threshold`
- `Q3_return_discipline`
- `Q4_return_ceiling`

`Q2_ew_departure` is demoted out of the main package because the reset results collapse to identical mixed-mid-cycle candidates and do not support a credible flagship visual.

## Style Direction

The package should read as institutional finance communication:

- restrained palette
- strong charcoal / neutral foundation
- one strong blue for selected scenarios
- muted red only for stress
- gold only when referring to gold
- large-scale readable typography
- generous whitespace
- no decorative infographic treatment
