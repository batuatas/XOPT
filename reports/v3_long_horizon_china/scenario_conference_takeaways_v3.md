# Scenario Conference Takeaways

- The main deck now uses four questions only, all centered on the locked robust 5Y benchmark.
- Soft landing is the strongest headline question because it is the clearest upside case without obvious instability.
- Higher-for-longer stays because it is the most intuitive adverse regime contrast.
- Gold more attractive stays because it is the only allocation-tilt question with clear visible movement.
- Return with breadth stays because it is the cleanest governance question: the benchmark can become broader, but not for free.
- US equities more attractive was removed from the main deck because the move is economically too small.
- The 6% / 7% / 10% return questions are appendix-only calibration checks.
- China remains secondary and belongs in appendix diagnostics, not the main scenario story.
